Atom Shortcuts

Move cursor

  • Ctrl+left - Move to beginning of word.
  • Ctrl+right - Move to end of word.
  • Home - Move to the first character of the current line.
  • End - Move to the end of the line.
  • Ctrl+Home - Move to the top of the file.
  • Ctrl+End - Move to the end of the file.
  • Ctrl+G - Move to a specific line. Row:column

Bookmarks

  • Ctrl+Win+Alt+F2 - Toogle bookmark on current line.
  • F2 - Jump to next bookmark.
  • Shift+F2 - Jump to previous bookmark.
  • Ctrl+F2 - Jump to bookmark.

Selections

Selections mirror many cursor movement commands but with the Shift key added.

  • Shift+Up - Select up.
  • Shift+Down - Select down.
  • Shift+Left - Select previous character.
  • Shift+Right - Select next character.
  • Ctrl+Shift+Left - Select to beginning of word.
  • Ctrl+Shift+Right - Select to end of word.
  • Shift+End - Select to end of line.
  • Shift+Home - Select to first character of line.
  • Ctrl+Shift+Home - Select to beginning of file.
  • Ctrl+Shift+End - Select to end of file.
  • Ctrl+A - Select entire contents of file.
  • Ctrl+L - Select the entire line.

Manipulation

  • Ctrl+J - Join next line to end of current line.
  • Ctrl+Up/Down - Move current line up or down.
  • Ctrl+Shift+DDuplicate current line.
  • Ctrl+K Ctrl+U - Uppercase the current word.
    • Ctrl+K Ctrl+L - Lowercase the current word.

Deleting

  • Ctrl+Shift+K - Delete current line.
  • Ctrl+Backspace Delete to beginning of word.
  • Ctrl+Delete - Delete to end of word.

Multiple cursors and selections

  • Ctrl+Click - Add a new cursor at each clicked location.
  • Alt+Win+Ctrl+UP/Down - Add a new cursor above/below the current cursor
  • Ctrl+D - Select the next word in the document that is the same as the currently selected word.
  • Alt+F3 - Select all words in the document that are the same as the currently selected word.

Brackets

  • Ctrl+M - Jump to the bracket matching the one adjacent to the cursor. It jumps to the nearest closing bracket when there's no adjacent bracket.
  • Alt+Ctrl+win+M - Select all the text in the current brakets.
  • Ctrl+Alt+. - Close the current XML/HTML tag.
